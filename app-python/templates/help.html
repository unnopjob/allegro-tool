Prerequisites

Before using this dashboard, make sure you have:

  • An Allegro Network Multimeter device accessible on your network
  • A valid Google Gemini API key (free at aistudio.google.com)
  • The FastAPI application running on your server

Quick Start (3 steps)

  1. Go to Settings and enter your Gemini API key, then click "Save & Test".
  2. Go to Devices, click "Add Device", fill in your Allegro device URL and credentials, then click "Activate".
  3. Return to Dashboard to see live network metrics.

Health Score

The health score (0–100) is calculated automatically:

  • Starts at 100
  • −10 for each DOWN interface
  • −5 for each interface with errors > 0
● 80–100: Healthy ● 50–79: Degraded ● 0–49: Critical

Auto-Refresh

Data refreshes automatically every 10 seconds. Click the "Refresh" button to refresh manually.

Top IPs Table

Shows the top 10 IP addresses by total bytes transferred, sorted in descending order.

The Analysis page uses Google Gemini to analyze your network data and generate insights.

Analysis Types

OverviewGeneral health summary of your network.
SecurityIdentify potential threats, suspicious traffic, or policy violations.
BandwidthFind top consumers and bandwidth utilization trends.
TCP FlowAnalyze TCP connections, retransmissions, and flow quality.
Root CauseDiagnose the root cause of network issues or anomalies.

Custom Questions

Optionally add a custom question to guide the analysis (e.g. "Why is there packet loss to 10.0.0.5?").

Found in the Analysis page, Path Check verifies connectivity between two hosts.

Results

  • Ping — Tests basic reachability and measures round-trip time.
  • Traceroute — Shows each hop along the path with IP address and RTT.
  • Port Status — Checks if a specific TCP/UDP port is open (shown when Port is specified).

Source IP and Port are optional. Only Destination IP is required.

Have a natural language conversation with the AI about your network. Responses are streamed in real time.

  • Press Enter to send a message.
  • Press Shift+Enter for a newline.
  • Click "Clear" to start a new session and erase history.

Chat sessions are identified by a UUID stored in your browser's localStorage.

Dashboard shows no data

  1. Verify a device is added and activated on the Devices page.
  2. Check that the device URL is correct and reachable.
  3. Use the "Test" button on the Devices page to verify connectivity.

AI Analysis not working

  1. Check that your Gemini API key is valid (Settings → Save & Test).
  2. Ensure your API key has quota remaining at aistudio.google.com.
  3. Verify the yellow warning banner is not shown on the Analysis page.

SSL / Certificate errors

If your Allegro device uses a self-signed certificate, uncheck "Verify SSL certificate" when adding the device.

File upload fails

  • Ensure the file is under 10 MB.
  • Only PDF, TXT, CSV, and JSON formats are supported.
  • Check browser console for detailed error messages.
